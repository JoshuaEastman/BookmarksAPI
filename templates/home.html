How to Use

Fetch approved bookmarks (supports ?tag, ?search, ?ordering):

GET localhost:8000/bookmarks/v1/bookmarks?tag=python&search=django&ordering=-created_at

Get a single bookmark by ID:

GET localhost:8000/bookmarks/v1/bookmarks/1/

Submit a new bookmark (anonymous, moderated):

POST localhost:8000/bookmarks/v1/bookmarks/submit/
Content-Type: application/json

{
  "title": "Great Django Guide",
  "url": "https://docs.djangoproject.com/",
  "description": "Official docs",
  "tags": ["django","docs"]
}

Unknown tags appear in pending_tags. Honeypot field website will be rejected if set.
